# 睇天气
#### 项目说明
天气预报小程序

#### 项目技术
微信小程序

#### 使用说明
1. 允许小程序获取当前位置，请求当前天气信息。
2. 点击首页顶部，进入详情页。
3. 上下滑动底部列表，查看未来天气信息。
4. 下拉页面，刷新当前信息。
